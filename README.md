# Drone Detection and Classification using YOLOv8

## Описание

Проект посвящен разработке системы обнаружения и классификации беспилотных летательных аппаратов (БПЛА) с использованием архитектуры YOLOv8.

Модель способна:

- обнаруживать БПЛА на изображениях;
- классифицировать тип БПЛА;
- отображать рамку вокруг объекта;
- работать через графический интерфейс.

---

## Используемые технологии

- Python 3.11
- YOLOv8
- PyTorch
- OpenCV
- Tkinter
